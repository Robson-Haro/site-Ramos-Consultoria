# Robson Ramos — Site institucional

## O que tem aqui
- `index.html` — o site completo (front), em um único arquivo. HTML, CSS e JavaScript puros, sem dependências externas além das fontes do Google Fonts.

## Como subir para o GitHub
1. Crie um repositório novo (ex: `robson-ramos-site`).
2. Coloque o arquivo `index.html` na raiz do repositório.
3. Suba (commit + push).

## Como publicar na Vercel
1. Entre em vercel.com e clique em "Add New Project".
2. Selecione o repositório do GitHub que você acabou de criar.
3. A Vercel detecta automaticamente que é um site estático — não precisa configurar nada (sem "build command", sem "output directory").
4. Clique em "Deploy". Em menos de 1 minuto o site estará no ar em uma URL da Vercel.
5. Depois, em "Settings > Domains", você conecta o seu domínio próprio.
